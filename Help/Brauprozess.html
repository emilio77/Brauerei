Brauprozess:

Der Brauprozess unterteilt sich in zwei Bildschirmdarstellungen:

  • Automatik

  • Manuell

Automatik:



  1. Zustandsanzeige für Brenner ( grün=an / rot=aus )

  2. Zustandsanzeige für Rührwerk ( grün=an / rot=aus )

  3. Zustandsanzeige für Pumpe ( grün=an / rot=aus )

  4. Zustandsanzeige für Brauautomatik ( grün=an / gelb= pause / rot=aus )

  5. Zustandsanzeige für Alarm ( grün=an / rot=aus )

  6. Heizung wird vor Erreichung der Zieltemperatur gepulst.Die Temperaturdifferenz bei welcher gepulst wird so wie die Pulsweite, kann in den Einstellungen angepasst werden.

  7. Heizung wird vor Erreichung der Zieltemperatur auf einen maximalen Gradienten geprüft. Ist der Gradient zu steil, wird die Heizung ausgeschaltet. Die Temperaturdifferenz bei welcher geprüft wird so wie der maximale Gradient, kann in den Einstellungen angepasst werden.

  8. Schalthysterese, kann in den Einstellungen angepasst werden.

  9. Sensorüberwachung – Verändert sich die log.txt während gestartetem Brauprozess binnen einer Minute nicht wird ein Alarm ausgegeben.

  10. Heizungsüberwachung – Sinkt die Temperatur trotz eingeschalteter Heizung um mehr als 2K wird ein Alarm ausgegeben.

  11. Alarm still, pulsen oder Dauersignal

  12. Starteinstellungen für (6)-(11), (20)-(24) und (29) speichern

  13. Brauprozess starten – Rechtsklick MULTISTART ← Details

  14. Brauprozess pausieren

  15. Brauprozess stoppen

  16. Timer gesteuerter Start des Brauprozesses ← Details

  17. Rast überspringen / zurück springen

  18. Rastzeit um eine Minute erhöhen / reduzieren

  19. Rast- und Brauerruf-Text-Eingabe. Mit dem Button wird auf die Ansicht zur Eingabe der Rast- und Brauerruf-Texte umgeschaltet. Dort kann pro Rast ein Rastname und ein Text eingegeben werde, der beim Brauerruf angezeigt wird. Die Texte werden mit dem Rezept gespeichert.

  20. Aktivierung für die einzelnen Rasten

  21. Ziel-Temperatur für Rast (Eingaben 0-100 erlaubt)

  22. Rastdauer für Rast (Eingaben 0-999 erlaubt, 0= Heizphase)

  23. Brauerruf am Ende der Rast (Alarm)

  24. Rührwerk für Rast verwenden inkl. Ein/Aus-Takt (Eingaben 0-999 erlaubt)

  25. Startbedingung für die Rastzeit:
    Feuer : Raststart bei Rasttemperaturüberschreitung
    Uhr: Raststart temperaturunabhängig
    Eis: Raststart bei Rasttemperaturunterschreitung

  26. Rast einfügen

  27. Rast löschen

  28. Rast nach unten schieben

  29. Rast nach oben schieben

  30. Umschaltung zwischen Manuell und Automatik Bildschirm

  31. Rezept laden (16)-(20)

  32. Rezept speichern (16)-(20)

  33. Umschaltung der Gesamt-Fortschrittsanzeige zwischen Gesamtprozesses, Gesamtrastzeit, Gesamtheizzeit und ETA (Expected Time of Abmaischen ) während des gestarteten Automatikbetriebs ( nicht editierbar)

  34. Grafische Ist-Temperatur-Anzeige
    Wurde die Temperaurmessung (z.B. Digitemp) über das Programm gestartet kann bei Ausfällen der Messung die Temperaturmessung durch Doppel-Click auf die Grafik erneut gestartet werden.

  35. Digitale Ist-Temperatur-Anzeige

  36. Digitale Soll-Temperatur-Anzeige

  37. Anzeige der Port-Nummer ( nicht editierbar)

  38. Anzeige der Port-Ausgabe ( nicht editierbar)


Manuell:



  1. Heizung manuell Ein/Aus
    Bei gestarteter Automatik zuerst auf manuellen Betrieb wechseln

  2. Rührwerk manuell Ein/Aus
    Bei gestarteter Automatik zuerst auf manuellen Betrieb wechseln

  3. Pumpe manuell Ein/Aus
    Bei gestarteter Automatik zuerst auf manuellen Betrieb wechseln

  4. Alarm manuell Ein/Aus
    Bei gestarteter Automatik zuerst auf manuellen Betrieb wechseln

  5. Simulation Ein/Aus
    Bei eingeschalteter Simulation wird die Temperaturdatei durch das Programm geschrieben. Hierzu kann ein Schieberegler zur Einstellung der Temperatur verwendet werden.

  6. Alle Rezepteinstellungen können auch im Manuellen Bildschirm verändert, gespeichert oder geladen werden. Details siehe Automatik.

  7. Umschaltung zwischen Manuell und Automatik Bildschirm.
